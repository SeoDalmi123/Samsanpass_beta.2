Notes Feature can help the user to be flexible in terms of saving important information. Users can save default information like any other notes application. In Sam San’s Pass case, we also have a calendar to remind the user additional information to be reminded about.
Notes Feature User interface is simple that can be used by anybody, in terms of neat design and accessible buttons, users can identify their tasks when it comes to using the notes feature.
Adding, Updating, and Deleting Notes is a must. While users can easily access those features, there are many options to do with the notes that are inputted in the notes. Users can add as many notes as possible and edit and update it while users can longer use the notes they can delete manually.
As simple as it shows, Users can be reminded about their upcoming events on the additional feature of the notes. Also, users can identify what time to that exact date and save every time they input a reminder and delete it afterwards.
